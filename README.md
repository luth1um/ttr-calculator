# 🏓 TTR Calculator

This project is a simple calculator for the Table Tennis Rating (TTR) created with React and TypeScript.
All calculations are done directly in the frontend, you don't need any backend to run this app.
In case you want to try out the calculator, just visit [this website](https://luth1um.github.io/ttr-calculator/).
In case you are only interested in the underlying TTR-calculator library, you should visit [this project](https://github.com/luth1um/ttr-calculator-typescript).

# 🚀 Quick Start

The usual commands for Node projects can be used in this project.

- `npm install` to install all dependencies
- `npm start` to run the app locally for development
- `npm format` to format the sources
- `npm lint` to run the linter
- `npm test` to run unit tests
- `npm e2e` to run E2E tests
- `npm build` to build the app for production
